===

## How to run the app

1\. Clone the repository:

```
git clone https://github.com/LukaszLasota/Lista-film-w
```

2\. Install the dependencies:

```
npm install
```

3\. Run dev server

```
npm start
```

4\. Build production version

```
npm run prod 
```
